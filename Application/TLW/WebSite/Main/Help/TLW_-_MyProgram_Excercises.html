Project: Tri-Living-Well

 

Module: Site.

 

Purpose:Exercise Page Detail (MyProgram_Exercises.aspx)

               

Site Area:

 

Input: QueryString[“programSeries”]

                #EXERCISE_ID > #801   (optional anchor tag)

 

 

  1. Get Data from SQL
    1. Query SQL by PROGRAM_SERIES
  2. Have the Webpage dynamically generate the page.
    1. Problem: Each exercise needs:

                                                               i.      To be included in a Table of Contents at the top.

                                                             ii.      To have its description below.

                                                            iii.      Note: Anchor tags should connect them and be based on EXERCISE_ID.

    1. Solution: RadGrids with ItemDataboud templates

                                                               i.      MyProgram_Excersies_1 = Table of Contents

                                                             ii.      MyProgram_Excersies_2 = description

 

Notes:

  • The Table of Contents and the descriptions will use a RadGrid and get layouts from template files in ~/HTML_Templates.
  • Example querystring request and also the Table of Contents links:
    1. MyProgram_Exercises.aspx?programSeries=800#804

                                                               i.      800 = PROGRAM_SERIES

                                                             ii.      804 = EXERCISE_ID
