TLW - User Registration Process

 

Starting a Program

 

Step 1

 

The user is greeted with information, and asked to take a health risk assessment.

Note: Currently only Framingham is working, we must decide how to handle Trale.

 

 

Popup of the Framingham assessment:

 

After taking the assessment and receiving a score, a user can input this score on step 2.
Step 2

 

The user enters the score they received during step 1, and clicks “Continue.”

 

 

The user is then presented with a fitness level, and must click on “This accurately describes me” to continue.

Note: We are currently defaulting everyone into the Well category.

 


 

Step 3

 

The user then can select a program from those listed, and the description below will adjust (Currently only the program id changes).  The user may also “See sample workouts” for the program they have selected.  Once they have made a decision and selected the appropriate program, they may “Begin Program.”
